# 九月二十六日然乌至波密

然乌到波密距离130公里是一个缓下坡，今天的骑行没有难度。

骑行没什么可说的，就是很舒服，在一个巨大的森林氧吧里头骑行。顺着帕隆藏布江而下，江水时缓时急，但水声一路可闻，清心养肺之骑啊。
